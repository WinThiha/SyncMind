## Why

The web app supports project member management, while Android only shows members read-only. Backend endpoints already support listing, adding/inviting, updating role/position, and removing members.

## What Changes

- Add Android API/repository support for project member list/add/update/remove.
- Add a project member management screen reachable from project detail.
- Support adding or inviting by email, changing role/position, and removing members.

## Impact

- Android project API service and repository
- Android navigation
- Android project detail and new member management UI
